01 — INTRODUCTION

Privacy Policy

SafeGram.one ("we," "our," or "us") operates the SafeGram mobile application (the "App"). This Privacy Policy explains how we collect, use, and protect your information when you use our App.

By using SafeGram, you agree to the collection and use of information as described in this policy.

02 — INFORMATION WE COLLECT

What We Collect

We collect the following types of information:

  • Account Information: Email address and password when you register
  • Device Information: Device model, OS version, unique device identifiers
  • Usage Data: Features used, scan history, app interactions
  • File Metadata: File names, sizes, and hash values for security scanning (not file content)
  • URL Data: URLs submitted for security analysis
  • Payment Information: Transaction reference names submitted for plan verification (no card numbers stored)
Important: We do NOT read the actual content of your personal files or messages. Only metadata and hash values are used for threat detection.
03 — HOW WE USE YOUR DATA

Use of Information

Your information is used to:

  • Provide and maintain the security scanning service
  • Detect malware, viruses, and malicious URLs
  • Process subscription payments and manage your account
  • Send important security alerts and service notifications
  • Improve detection accuracy and app performance
  • Comply with legal obligations
04 — ACCESSIBILITY SERVICE

Accessibility Permission

SafeGram uses Android's Accessibility Service to monitor links and files shared through messaging applications (Telegram, WhatsApp, and others) for real-time threat detection.

Scope: This permission is used exclusively for security scanning. We do not read, store, or transmit the content of your private messages. The service only inspects URLs and file references for known threat signatures.

You may disable this permission at any time through Android Settings → Accessibility → SafeGram. Disabling it will turn off real-time messenger protection.

05 — THIRD-PARTY SERVICES

Third Parties

We use the following third-party services:

  • VirusTotal API — File hash and URL threat scanning
  • Firebase (Google) — Authentication and database storage

These services have their own privacy policies. We recommend reviewing them. We do not sell your data to any third party.

06 — DATA RETENTION & SECURITY

Security & Retention

We implement industry-standard security measures including encrypted data transmission (HTTPS/TLS), Firebase security rules restricting data access, and regular security reviews.

Your data is retained for as long as your account is active. You may request deletion of your account and associated data by contacting us.
